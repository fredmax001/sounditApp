## ΏθΡδ CRITICAL BUGS FOUND & FIXED DURING VALIDATION

### Bug 1: `/events/saved` returned 422 instead of 401
- **Root cause:** `/{event_id}` route was registered before `/saved`, so "saved" was parsed as an integer event ID
- **Fix:** Moved `@router.get("/saved")` to appear before `/{event_id}` in `api/events.py`
- **Verification:** Now returns 401 for unauthenticated requests έεΖ

### Bug 2: Ticket scanner validation returned 422
- **Root cause:** `validate_ticket` endpoint expected `ticket_code` as query param, but `Scanner.tsx` sends it as `application/x-www-form-urlencoded` body
- **Fix:** Changed parameter to `ticket_code: str = Form(...)` in `api/ticketing_organizer.py`
- **Verification:** Form-data POST now returns 404 for invalid tickets έεΖ
